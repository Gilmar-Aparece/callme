# Nova Interview (No API needed)

This version runs 100% in the browser — no internet, no API key, no server.

## What changed
- Removed all calls to the Anthropic API.
- Nova now uses a built-in question bank with randomized icebreakers, technical
  questions (matched against keywords in your resume — PHP, JavaScript, React,
  Vue, SQL, Docker, AWS, Git, etc.), teamwork/collaboration questions, debugging
  questions (always followed by "how long did that take?"), project timeline
  questions, and HR/behavioral questions.
- Reactions before each new question are randomized so it doesn't feel repetitive.
- Voice input/output still works fully offline via the browser's built-in
  Web Speech API (Chrome recommended).

## How to use
Just open `index.html` in Chrome. No setup, no server, no API key.
